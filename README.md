# 🐝 Aventuras no Jardim - Mel's Garden Adventures

An interactive, engaging educational game designed to teach mathematics concepts to children through fun, garden-themed challenges. Help Mel the bee complete 6 missions by solving math puzzles!

**🌐 Live Demo:** https://aventurasnojardim.vercel.app

---

## 🎮 About the Game

**Aventuras no Jardim** (Garden Adventures) is a delightful educational platform where players guide Mel the bee through a magical garden while mastering fundamental math skills. The game features colorful animations, interactive challenges, and a rewarding sticker collection system.

### 📚 Learning Outcomes
Players develop skills in:
- **Counting & Number Recognition** - Matching quantities to digits
- **Addition** - Combining groups of objects
- **Subtraction** - Removing items and finding the remainder
- **Spatial Reasoning** - Programming bee movement on a grid
- **Pattern Recognition** - Identifying and completing sequences

---

## 🎯 Game Features

### 6 Educational Phases

1. **🔗 Linking & Counting (Phase 1)** - Link groups of objects to their corresponding numbers
2. **🐝 Walking in the Garden (Phase 2)** - Program Mel's movement on a grid using directional commands
3. **🍄 Subtracting Mushrooms (Phase 3)** - Remove items and count what remains
4. **🌸 Among the Flowers (Phase 4)** - Count flowers and select the correct number
5. **🐞 Adding Ladybugs (Phase 5)** - Combine groups and reach target totals
6. **✨ Magic Pattern (Phase 6)** - Identify patterns and complete sequences

### ✨ Interactive Features
- 🎫 Sticker Collection System with personal notebook
- 💡 Context-sensitive hints for each challenge type
- 🔊 Portuguese (pt-BR) text-to-speech narration
- 📱 Responsive mobile-friendly design
- 🎉 Celebration effects and animations
- 👤 Personalized experience with name saving

---

## 🛠️ Technology Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Styling:** Tailwind CSS 3.4.17 + Custom CSS animations
- **Typography:** DM Sans (Google Fonts)
- **Data Persistence:** Browser LocalStorage with SDK support
- **Text-to-Speech:** Web Speech API

## 📂 Project Structure

🎮 Gameplay
Challenge Types
Challenge	Objective	Skills
🔗 Linking	Match object groups to numbers	Counting, number recognition
🐝 Path	Program bee movement to reach flowers	Spatial reasoning, planning
🍄 Subtraction	Remove items and count remainder	Subtraction, counting
🌸 Counting	Select correct quantity	Number recognition
🐞 Addition	Combine groups to reach total	Addition, mental math
✨ Pattern	Complete emoji sequences	Pattern recognition, logic
💾 Data & Customization
LocalStorage Keys
bee_player_name - Saved player name
bee_game_progress - Challenge completion records
Customizing Challenges
Edit the challenge functions in script.js:

buildLinking() - Object types and quantities
buildSubtraction() - Initial amounts and removals
buildCounting() - Target numbers
buildAddition() - Number combinations
buildPattern() - Emoji sequences
